# wst-eccentricity

Rapid classification of eccentric compact binaries with the **wavelet
scattering transform** (WST).

This package accompanies *"The Shape of Eccentricity: Rapid Classification of
Eccentric Binaries with the Wavelet Scattering Transform"* (Canizares, Staelens
& Romero-Shaw, University of Cambridge). It computes the WST of pre-generated
gravitational-wave strain data and trains a compact 1D-CNN to distinguish
eccentric from quasi-circular signals.

Waveform generation is out of scope: the pipeline starts from a provided
waveform dataset and its parameter files.
